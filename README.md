# Personal Expense Tracker

A full-stack Python application built with Flask, SQLAlchemy (SQLite), and Chart.js to help users track their monthly income and expenses with visual analytics.

## Features

- **Monthly Budgeting**: Set a salary/income for specific months.
- **Expense Categorization**: Log expenses with categories (Food, Rent, Bills, etc.).
- **Dynamic Dashboard**: Real-time updates of Total Spent, Balance, and Budget Usage percentage.
- **Visual Analytics**: Interactive doughnut charts showing spending distribution by category.
- **History Management**: View and delete past transactions for the selected month.
- **Persistent Storage**: Data is saved locally in a SQLite database.
- **Mobile Responsive**: Clean UI built with Bootstrap 5.

## Prerequisites

- Python 3.8 or higher
- `pip` (Python package installer)
